## 2026-01-08 - CLI Colorization
**Learning:** Adding ANSI colors to a CLI tool's logging output significantly improves readability and status monitoring, especially for different log levels (INFO vs ERROR).
**Action:** Use a custom `logging.Formatter` to inject color codes safely without affecting file logs.
